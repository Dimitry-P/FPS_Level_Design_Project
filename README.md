# First-Person Level Design – Game Design Focus

## Overview
This Unity project demonstrates a fully playable first-person level created entirely without using any C# scripts.  
The focus of the project is **game design**, interactive gameplay, and visual layout, rather than programming.

## Features
- Multiple rooms with unique designs and materials
- Interactive hazards (spikes that rise and fall)
- Moving platforms to navigate
- Collectible coins
- Teleporters to move between areas
- Distinct wall materials and colors to differentiate rooms

## Purpose
The goal of this project was to showcase **level design skills** in Unity:
- Creating a playable environment from scratch
- Designing engaging challenges for the player
- Using materials, colors, and layout to enhance the gameplay experience

## How to Play
- Player controls a first-person character
- Navigate through obstacles and hazards
- Collect coins scattered throughout the level
- Use moving platforms and teleporters to reach new areas
- Avoid spikes and other deadly traps

## Notes
- No scripts or programming were used — all interactions are created using Unity’s built-in components.
- This project emphasizes creativity, design thinking, and interactive environment setup.
